# Luật Schedule

Ứng dụng quản lý lịch học đồng bộ cho sinh viên Đại Học Luật TPHCM.

## 🚀 Tính Năng

- 📅 Xem toàn bộ lịch học ở một nơi
- 🔄 Đồng bộ hóa tự động với Google Sheets  
- 🛡️ Bảo mật cao với xác thực
- ⚡ Hiệu năng nhanh
- 📱 Responsive cho mọi thiết bị
